# 🌿 Gardon Of King (植物大战僵尸：国王的花园)

> 我的第一个 Unity 项目。

---

## 📝 项目简介

这是我的第一个 Unity 项目。

在开始这个项目的时候，我**几乎没有任何游戏开发经验**。  
而且由于视频热度去的快，我需要在极短时间内学会Unity游戏开发并发布第一个试玩DEMO。  
很多东西都是一边摸索一边完成的，项目也做得很仓促,  
甚至连Garden都拼成了Gardon，好在影响不大:)  

项目中难免存在：  
- 结构混乱
- 实现方式原始
- 一些看来不合理的设计

但它记录了我从 0 开始学习 Unity 的过程，也承载了很多尝试与思考。

---

## ⚠️ 项目状态

由于一些个人原因，这个项目目前已经**停止更新**。

与其让它沉寂下去，我选择将它**开源出来**。

---

## 🔥 关于这个项目的一点期待

如果你刚好对这个项目感兴趣，或者觉得它还有发展的空间：

你可以自由地：

- 继续开发它
- 重构已有系统
- 修改或扩展玩法

也许它本身并不完善，  
但如果有人愿意继续走下去，它仍然可以变得更好。

> 希望它不只是一个被遗忘的练习项目，  
> 而是一点还能被延续的火种。
